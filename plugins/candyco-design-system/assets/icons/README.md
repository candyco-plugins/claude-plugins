# Icons — Lucide

CandyCo uses [Lucide](https://lucide.dev) for all UI icons. Loaded from CDN; no local files required.

```html
<script src="https://unpkg.com/lucide@latest/dist/umd/lucide.min.js"></script>
<script>lucide.createIcons();</script>
```

Or in React:

```jsx
import { Factory, Truck, Package } from 'lucide-react';
```

**Stroke:** 1.5px (default). **Color:** `currentColor` — never set explicitly.
**Substitution flag:** Lucide is the chosen substitute; if CandyCo has a proprietary icon library, please share it.
