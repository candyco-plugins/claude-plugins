# Complaint response template — consumer complaint investigation

CandyCo's standard written response to a retail or brand customer (Costco,
Sam's Club, Hershey, Trader Joe's, Walmart, Kroger, and co-manufacturing
partners) about a consumer complaint investigation. Built on the paged 8.5×11
report layout and the structure GFSI-certified customers expect: identification
and lot traceability, immediate containment, evidence-based investigation
(production records, retained samples, complaint history), a named root cause
method, scope determination, product disposition, and time-bound corrective
actions with verification of effectiveness.

## Files

```
complaint-response-template/
├── README.md       ← this file
└── template.html   ← starter HTML — copy and fill the yellow fields
```

## What's inside template.html

- **Pages 1–4 (customer-facing):** cover + product identification + complaint
  restated verbatim + conclusion hero; containment + investigation detail;
  root cause analysis + conclusion and scope + disposition; CAPA table +
  closing + signature block.
- **Internal pages A–B (delete before sending):** how to use the template,
  response timelines (acknowledge in 24 hours, full report in 14 calendar days
  or the customer's SLA), severity classification, confectionery-specific
  notes, writing do's and don'ts, and a pre-send checklist.

## How to start a new response

1. Copy `template.html` to a working folder and rename it (e.g.
   `CC-2026-0412-costco-response.html`).
2. Copy `candyco-logo-black.png` from `../assets/logo/` into the same folder
   and update the `<img src>` to the local filename.
3. Fill every yellow-tinted field; delete optional rows and blocks that don't
   apply. No yellow may remain at send time.
4. Delete both internal pages and renumber footers if pages were added or
   removed (the customer report is Page N of 4 by default).
5. Run the pre-send checklist on internal page B, then print to PDF (Letter,
   portrait, background graphics on, browser headers/footers off).

## Print-fit check

Same rule as every paged report: each sheet must fit one Letter page. If a
section grows past the page (long CAPA tables, multi-lot dispositions), split
it onto a new sheet — do not shrink fonts. Verify by printing to PDF and
confirming the page count equals the sheet count.

## Sources

Structure and timelines per Walmart supplier complaint-investigation
expectations, Costco Food Safety & Quality Audit Expectations V3, SQF complaint
management (2.1.4), and BRCGS Issue 9 clause 3.10. Tone and legal guidance
(goodwill replacement is not an admission of fault; legal review on illness or
injury claims) reflect industry practice, not legal advice.
